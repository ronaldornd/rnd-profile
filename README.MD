# RND Profile

Este é um projeto React.js escrito em TypeScript para criar um perfil pessoal. Ele inclui uma página de demonstração hospedada em ronaldornd.vercel.app.

## Exemplo

Confira um exemplo ativo do projeto em HTTPS://ronaldornd.vercel.app.

## Instalação

1. Clone este repositório:
git clone https://github.com/ronaldornd/rnd-profile.git

2. Navegue até o diretório do projeto:
cd rnd-profile

3. Instale as dependências:

npm install


## Uso

Após instalar as dependências, você pode iniciar o servidor de desenvolvimento local:

npm run dev

Isso iniciará um servidor local em http://localhost:3000, onde você poderá visualizar e editar no vs code para adicionar suas informações.

## Estrutura do Projeto

A estrutura do projeto segue um padrão comum para aplicativos React.js:

- src/: Contém todo o código-fonte do aplicativo.
- components/: Componentes React reutilizáveis.
- pages/: Páginas React que são renderizadas em diferentes rotas.
- public/: Contém arquivos estáticos, como imagens ou fontes.
